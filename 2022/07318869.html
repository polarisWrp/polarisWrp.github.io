Js闭包例子补充

JavaScript
发布日期:   2022-07-31
文章字数:   763
阅读时长:   3 分

1. 闭包的定义

  • 一个函数和对其周围状态（lexical environment，词法环境）的引用捆绑在一起（或者说函数被引用包围），这样的组合就是闭包（closure） ;
  • 闭包让你可以在一个内层函数中访问到其外层函数的作用域;
  • 在 JavaScript 中，每当创建一个函数，闭包就会在函数创建的同时被创建出来。

1.1 栗子1形如：

<script>
  function init() {
    // name 是一个被 init 创建的局部变量
    var name = "Mozilla"; 
    // displayName() 是内部函数，一个闭包
    function displayName() { 
      // 使用了父函数中声明的变量
      console.log(name); 
    }
    displayName();
  }
</script>

1.2 栗子1改写：

  • 变量name依旧可以访问的原因？
    1. js中的函数会形成闭包；
    2. 闭包是由函数以及声明该函数的词法环境组合而成，这个环境包含了这个闭包创建时作用域内的任何局部变量；
    3. 栗子中，变量myFn 是执行makeFunc函数时创建的displayName函数实例的引用，而displayName函数实例维持了一个对它的词法环境（变量name在其中）的引用
    4. 所以当 myFn 被调用时，name属性依旧可以访问
<script>
  function makeFunc() {
      var name = "Mozilla";
      // 闭包函数 displayName
      function displayName() {
        console.log('2222', name);
      }
      return displayName;
  }

  var myFn = makeFunc();
  myFn();
</script>

1.3 栗子2:

  • 定义一个函数makeAdder,该函数接收一个参数 x，返回一个接收y参数的匿名函数，这个匿名函数返回x+y
<script>
  // 本质上 makeAdder是一个工厂函数，它创建了将指定的值和他的参数相加求和的函数
  function makeAdder(x) {
    return function(y) {
      return x + y;
    };
  }
  // add5 和 add10 都是闭包。它们共享相同的函数定义，但是保存了不同的词法环境。
  // 在 add5 的环境中，x 为 5。而在 add10 中，x 则为 10。
  var add5 = makeAdder(5);
  var add10 = makeAdder(10);

  console.log('55555555', add5, add5(2));  // 7
  console.log('101010101010', add10, add10(2)); // 12

</script>

1.4 使用闭包模拟私有方法

  • 定义函数思路：
    1. 使用闭包定义公共函数，并且使其可以访问私有函数和变量;
    2. 闭包函数只定义了一个词法环境，为三个函数所共用;
      2.1 环境中存在两个私有项，变量privateCounter和方法changVal
      2.2 这两个私有项外部无法访问，只能通过定义的三个匿名函数访问
    3. 请注意两个计数器 Counter1 和 Counter2 是如何维护它们各自的独立性的;
    4. 每个闭包都是引用自己词法作用域内的变量 privateCounter ;
    5. 每次调用其中一个计数器时，通过改变这个变量的值，会改变这个闭包的词法环境;
    6. 然而在一个闭包内对变量的修改，不会影响到另外一个闭包中的变量。
<script>
  var Counter = (function() {
    var privateCounter = 0
    function changVal(val) {
      privateCounter += val
    }
    return {
      increment: function() {
        changVal(1)
      },
      decrement: function() {
        changVal(-1)
      },
      value: function() {
        return privateCounter
      }
    }
  })()

  console.log(Counter.value()); /* logs 0 */
  Counter.increment();
  Counter.increment();
  console.log(Counter.value()); /* logs 2 */
  Counter.decrement();
  console.log(Counter.value()); /* logs 1 */

</script>
